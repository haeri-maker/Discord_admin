GitHub에서 데이터 파일로 관리하는 방법

매번 Claude에게 요청하지 않고, GitHub에서 data.js 파일만 수정하면 자동으로 업데이트됩니다.

1
GitHub repository에서 New file 클릭 → 파일명 data.js 로 생성
2
아래 형식으로 신청자/제출 데이터 작성 후 Commit:
window.EXTRA_APPLICANTS = [{email:"...",name:"...",handle:"...",country:"...",challenge:3}];
window.EXTRA_SUBMISSIONS = [{email:"...",name:"...",handle:"...",url:"...",challenge:3}];
3
index.html 의 </body> 태그 바로 위에 <script src="data.js"></script> 추가 → 이후부터 data.js만 수정하면 자동 반영!
4
또는 Claude에게 새 신청자/제출 PDF를 주면 업데이트된 index.html을 바로 만들어드립니다 😊
